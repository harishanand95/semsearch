Question Answer Platform

Academic Question Answer Platform aims at providing teachers and students a comprehensive and interactive discussion platform. This project is an implementation of community based question answering services. It has a provision for teachers and students to post answers to an already existing question, upvote an answer, search for previously posted questions, upload files and allows verification of answers by the teachers. The questions posted will be categorised topic-wise.
